## BiggerBox 是个什么套餐

BiggerBox 的全名叫 **BIGGERBOX-20-1024**，是搬瓦工从 2024 年底开始推出的 Box 系列限量版套餐之一，目前属于"抢到即是赚到"的那类产品。

配置如下：

- **CPU**：1 核 Intel Xeon
- **内存**：1024 MB（1 GB）
- **硬盘**：20 GB SSD RAID-10
- **月流量**：1000 GB（1 TB）
- **带宽**：1 Gbps
- **机房**：美国洛杉矶 DC99（USCA\_99）
- **线路**：三网回程 CN2 GIA（电信 / 联通 / 移动均走 CN2 GIA 高端骨干）
- **价格**：$37/年（约合 270 元人民币）

它的上游是搬瓦工的 DC99 机房——一个用来做灰度测试的洛杉矶节点，物理位置和知名的 DC9 同址，但走的是 CN2 GIA 三网回程，性能表现和 DC9 相差无几。

和 Box 系列的前一款 MiniBox（$27/年，512MB 内存 / 10GB / 500GB 流量）相比，BiggerBox 内存翻倍、硬盘翻倍、流量翻倍，价格只多了 $10，明显更划算。

BiggerBox 买到之后也可以在 KiwiVM 后台直接补差价升级到 PowerBox（$45/年，1.5GB 内存 / 30GB / 1.5TB 流量），整个 Box 基础系列形成一个完整的升级阶梯。

## 搬瓦工限量版 vs 常规套餐，到底差在哪

很多刚接触搬瓦工的朋友会好奇：为什么限量版套餐比常规套餐便宜这么多？官方有没有偷偷缩水什么东西？

其实没有太多神秘的地方。

搬瓦工推出限量版套餐的逻辑，有点像超市里的"特供款"——用固定机房、不保证 SLA 在线时间（官方标注 best effort）、不支持机房迁移，换来更低的价格门槛。常规套餐则可以在十几个机房之间随时切换，有在线时间保证，服务更完整，所以贵一些。

对大多数个人用户来说，Box 系列那个"无 SLA 保证"其实影响不大——搬瓦工经营多年，整体稳定性有口皆碑，DC99 / DC1 机房的实际跑分和正式机房差异很小。实测电信晚高峰能跑 800-900 Mbps，用来跑代理、轻量博客、日常工具完全够用。

真正需要多机房切换灵活性，或者对线路稳定性有极端要求（比如跑生产业务）的，才建议考虑 CN2 GIA-E 常规套餐。
